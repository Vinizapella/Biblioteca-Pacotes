# 📚 Biblioteca de Pacotes

> Exploração e organização de pacotes e bibliotecas — reutilização de código de forma inteligente.

---

## 🗂️ Sobre

O projeto **Biblioteca de Pacotes** tem como objetivo estudar e demonstrar o uso de **pacotes** em Java, organizando classes em estruturas coesas, reutilizáveis e bem definidas — seguindo boas práticas de modularização.

---

## 📖 Conteúdo abordado

- 📦 Criação e organização de pacotes (`package`)
- 🔗 Importação de pacotes (`import`)
- 🏗️ Separação de responsabilidades por pacote
- ♻️ Reutilização de classes entre pacotes
- 🔐 Modificadores de acesso (`public`, `private`, `protected`)

---

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/Vinizapella/Biblioteca-Pacotes.git

# Acesse a pasta
cd Biblioteca-Pacotes

# Compile e execute
# (ajuste conforme o ambiente utilizado)
javac src/main/java/Main.java
java Main
```
